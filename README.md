# MvvmImageClient（Android 客户端）

本仓库是 **安卓手机端** 的开源小工具：在局域网内连接电脑上运行的「文件夹图片服务」，浏览对方已选中的图片，支持大图缩放拖动，并把勾选的图片 **保存到手机相册**。

电脑端需要单独运行对应的桌面服务（任选其一即可：你自己的实现，或与本客户端配套的桌面项目）。手机与电脑需处于 **同一 Wi‑Fi**。

## 适合什么场景？

- 想在 **手机大屏** 上看电脑里的一批图，少一步「先发到聊天再保存」。
- 想从列表里 **挑几张** 存进相册，其余保持不动。

---

## 第一次怎么用？

1. **在电脑上** 启动文件夹图片服务：选好目录、在桌面端勾选要分享的图片，点击 **启动服务**，记下界面上的 **IP 地址**（如 `192.168.x.x`）和 **端口**（如 `8081`）。
2. **在手机上** 安装本应用，点右上角 **扳手图标**，把地址和端口填进去并 **保存**。
3. 回到列表 **下拉刷新**，即可看到电脑端当前勾选的那批图。

> **提示**：地址或端口填错、电脑没开服务、或不在同一网络，列表可能一直为空或加载失败。

## 界面说明（按按钮看）

### 顶部工具栏

| 按钮 | 作用 |
|------|------|
| **扳手图标** | 打开连接设置，填写电脑的 IP 与端口。 |
| **全选 / 取消全选** | 首次点击：选中当前列表全部；再次点击：取消当前列表的全部选中。 |
| **下载到相册** | 将当前 **已勾选** 的图片写入相册（常见路径：**图片 → MvvmImageClient**，以各机型相册为准）。 |

### 列表每一行

- **行末勾选框**：选中 / 取消这一张（与「全选」共用同一套选择）。
- **点击行主体（缩略图或文字）**：进入大图；双指缩放、单指拖动；再点一下图片可隐藏或显示顶部栏。大图页右上角的 **选择 / 已选** 与列表勾选 **实时同步**。

### 下拉刷新

从列表顶部向下拉，重新从服务端获取当前勾选列表（电脑端改了勾选后，手机刷新即可同步）。

## 常见问题

- **列表为空？** 检查电脑服务是否运行、IP/端口是否与电脑端一致、是否同一 Wi‑Fi。  
- **相册里找不到刚保存的图？** 在相册中查看 **MvvmImageClient** 相册，或按「最近添加」排序。  
- **用 Android 模拟器连本机服务？** 通常需使用模拟器访问宿主机的专用地址（例如 `10.0.2.2`），具体以官方模拟器文档为准。

---

## 技术栈与本地构建

- **Kotlin**，**MVVM**，**Room**，**Retrofit**，**OkHttp**，**Hilt**，**Glide**，**PhotoView**  
- 应用内配置的 IP/端口通过 **OkHttp 拦截器** 动态替换请求主机，便于局域网调试。

克隆仓库后，使用 **Android Studio** 打开根目录，连接设备或模拟器，执行 **Run** 即可。需 **JDK 17** 与项目自带的 Gradle Wrapper。

若你维护配套的桌面服务端仓库，欢迎在 Issue 或本 README 中补充 **服务端项目链接** 与 API 约定，方便他人一键对照使用。
